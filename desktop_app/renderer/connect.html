Connection Flow

  1. Connect by cable to the robot service network and open the advanced configuration panel.
  2. Refresh wireless networks, choose the preferred Wi‑Fi, enter the password, then connect.
  3. Refresh again after success and note the assigned wireless IP.
  4. Disconnect the cable, switch your PC to the same network, then reconnect to that IP from this manager.
  5. Connection is confirmed when localization, network delay, map, device name, and note are visible on the device card.
Pick a robot to load wireless status.

Special Network Notes

If the robot uses an external wireless client, make sure the client’s Ethernet output is plugged into the controller port assigned for uplink traffic after the client joins the field network.

For DHCP mode, remember the assigned wireless IP and reconnect to that address from the manager after you leave the service network.

For static mode, enter the assigned IP, subnet mask, and gateway, apply them, then reconnect using the new static address.
